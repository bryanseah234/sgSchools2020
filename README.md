# sgSchools2020

A comprehensive directory of Singapore schools displayed on a map and as a clickable web list.

## Description

This project provides a comprehensive directory of Singapore schools including Primary Schools, Secondary Schools, Junior Colleges, Polytechnics, and ITE. The schools are presented in two formats: an interactive Google Maps visualization and a web-based HTML listing with direct links to each school's official website.

## Features

- Interactive Google Maps showing locations of all Singapore schools
- Clickable HTML directory with links to official school websites
- Coverage of all educational levels (Primary, Secondary, JC, Polytechnic, ITE)
- Simple, static web page for easy access and fast loading

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Maps (for map visualization)
- Glitch (for hosting the web listing)

## Installation

```bash
# Clone the repository
git clone https://github.com/bryanseah234/sgSchools2020.git

# Navigate to project directory
cd sgSchools2020
```

No additional dependencies required - simply open `schools.html` in a web browser.

## Usage

```bash
# Open the HTML file in your browser
open schools.html
# or on Windows
start schools.html
```
